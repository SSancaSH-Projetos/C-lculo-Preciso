Funcionalidades

Funcionalidades

A nossa calculadora é uma forma de auxiliar operadores nos calculos de orçamentos na área de usinagem.

Certificações

Certificações

A nossa implementação sera com base no que os alunos da usinagem aprendem conforme o curso, nos ultilizaremos as peças deles para nos certificarmos de que nossos calculos foram 100% precisos, com a nossa ideia nós iremos produzir este software a partir de três linguagens, Java como back-end, React Native e JS como front-end e SQL como banco de dados.

Disponibilidades

Disponibilidades

Nós disbilizaremos uma interface mobile e um site, para que qualquer pessoa possa acessar, nosso desenvolvimento começará no final de janeiro e estará finalizado no final de maio.
